College Event Registration Webpage
Project Overview

The College Event Registration Webpage is a simple front-end web application that allows students to register for different college events.
Users can enter their name and select an event from the dropdown list. After submitting the form, a confirmation message is displayed.

This project is built using HTML, CSS, and JavaScript.

Features
Simple and user-friendly interface
Event selection dropdown
Stylish design using CSS
Icons using Font Awesome
Button hover animation
Form submission handled with JavaScript
Displays confirmation message after registration
Technologies Used
HTML – For webpage structure
CSS – For styling and layout
JavaScript – For form functionality
Font Awesome – For icons
How It Works
The user enters their Full Name.
The user selects an event from the dropdown menu.
When the Register button is clicked:
JavaScript prevents the page from refreshing.
The entered name and selected event are captured.
A confirmation message is displayed on the page.

Example output message:
